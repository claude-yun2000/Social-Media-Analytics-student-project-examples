# Social Media Anylytics course: sample student projects

With the consent of the students, this repository contains a couple of project presentations (slides in PDF format containing Python codes) by our undergraduates in the Department of Communication for the Social Media Analytics course:
- Analysis of user comments of the Apple iOS 18 on Reddit
- Readers' comments on ethnic Yao Chinese folktales on Bilibili

Since Github does't support the preview of PDF files, you will need to fork (download) the PDF files to read them.
